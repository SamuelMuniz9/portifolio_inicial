# Samuel Muniz - Portfolio 

## 🎨 Portfólio Profissional de Samuel Muniz

Um portfólio moderno, responsivo e visualmente atraente construído inspirado no design minimalista do Keita Yamada.


### 🎯 Seções do Portfólio

1. **Hero Section**: Apresentação impactante com call-to-action
2. **Sobre**: Informações sobre Samuel, especialidades e tech stack
3. **Projetos**: Galeria dos 4 principais projetos com detalhes
4. **Contato**: Opções de comunicação (WhatsApp, Email, Social Media)

### 💻 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica moderna
- **Tailwind CSS**: Framework de utilidade CSS
- **JavaScript Puro**: Interatividade sem frameworks pesados
- **GSAP**: Animações suaves e ScrollTrigger
- **Lucide Icons**: Ícones SVG otimizados
- **Vercel**: Hosting e deployment

### 📱 Responsividade

O site é totalmente responsivo com breakpoints:
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
